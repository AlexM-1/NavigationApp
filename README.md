# NavigationApp

Мобильное приложение для социальной сети. Авторизация, просмотр ленты новостей, страница пользователя, раздел избранное.

[**Проект Figma**](https://www.figma.com/file/s1N2cHHSmLScPf5dhDX8Oc/мобильные-приложения?node-id=724%3A2347)


### Implemented:
- Authorization by sms 
- Scrolling and Zooming Photos
- Add photos from gallery
- MVVM pattern + coordinator
- Группировка постов в ленте по дате
- Загрузка данных по сети или из локальной модели
- Caching images
- Сохранение избранных постов (core data)
- Темная тема
- Локализация rus/eng


## Skills
* UIKit, Auto Layout
* URLSession, Rest API, JSON, Codable
* Concurrency
* CoreData
* MVVM + coordinator
* Figma
